# 分割线组件规范（Figma 设计稿标准）

## 文档说明

本规范根据 Figma 设计稿 **Divider Guidelines**（测试V5.1 Web-H5 组件库）中的分割线组件**逐项提取**，描述线宽、线色、方向、尺寸、使用原则、正确/错误场景及极端原则等所有可量化与可实现的属性，供开发与设计对齐使用。

**设计稿来源：**

## 1. 设计原则（使用原则）

### 1.1 三条核心原则

| 原则 | 说明 |
|------|------|
| **信息区隔** | 多用于区隔文字信息。 |
| **区分主次** | 使用时尽量在大段文本内使用，以文本内容为重点。 |
| **降低理解难度** | 用来拆分大段内容，降低文本传递给用户的难度。 |

- 分割线是**辅助性**组件，不应抢夺内容焦点；应用场景以**段落、章节、信息块**之间的视觉分隔为主。

---

## 2. 基础样式定义

### 2.1 线宽（Stroke / Size）

| 属性 | 取值 | 说明 |
|------|------|------|
| **线宽** | `1px` | 所有分割线、小竖线统一为 1px，无其他档位。 |
| **实现** | `height: 1px`（水平）/ `width: 1px`（垂直） | 与设计稿「Details」中「分割线（1px）」「小竖线（1px）」一致。 |

### 2.2 颜色与透明度

**水平分割线（分割线）：**

| 属性 | 取值 | 说明 |
|------|------|------|
| **默认线色** | 设计令牌 `--color-line-border-subtle` | 规范中用于分割线、文档标题下分割线等。 |
| **色值 Fallback** | `#f2f3f4`（浅灰） | 与设计稿中 Level 1/Light 及边框色一致。 |
| **其他参考** | `#E8EBEB` | 部分设计稿标注用浅灰，实现时以设计令牌优先。 |
| **不透明度** | `100%` | 默认不透明，无半透明分割线变体。 |

**小竖线（垂直）：**

| 属性 | 取值 | 说明 |
|------|------|------|
| **线色** | `#C4C7CA` | 取自 Figma Divider-Web 小竖线资产（Line 338，节点 7136:55684），与水平分割线不同。 |
| **实现** | `background: #C4C7CA` 或 `--color-line-divider-vertical` | 小竖线**不可**使用水平分割线的 `--color-line-border-subtle`，需单独指定。 |

- 若设计系统存在 **Primary / Secondary / Faded / Highlight / Critical** 等类型（见设计稿「类型」列表），线色由对应语义色或边框 token 决定，线宽仍为 1px。

### 2.3 形状与方向

| 属性 | 说明 |
|------|------|
| **线型** | 实线（Solid），无虚线、点线等变体。 |
| **方向** | 默认**水平**；垂直形态为「小竖线」变体，见 §3。 |

## 3. 组件类型与尺寸

### 3.1 两种展示类型

设计稿中 **Divider-Web** 组件明确区分两种类型：

| 类型 | 名称 | 方向 | 宽度 | 高度 | 说明 |
|------|------|------|------|------|------|
| **分割线** | Type=分割线 | 水平 | 自适应（见下表） | `1px` | 用于段落、区块之间的水平分隔。 |
| **小竖线** | 小竖线 | 垂直 | `1px` | `16px` | 用于列表、导航等垂直分隔，由 1px 线旋转 90° 实现。 |

### 3.2 分割线（水平）尺寸

| 场景 | 宽度 | 说明 |
|------|------|------|
| **默认展示** | `335px` / `343px` | 设计稿 Divider_V5、Divider-Web 示例宽度，可作为默认或窄屏参考。 |
| **内容区通栏** | `100%` 或父容器宽 | 在正文容器内用于分隔章节时，与内容区同宽（如 1154px 示例）。 |
| **实现** | 由父容器或 `width` 指定 | 不固定死宽度，随布局自适应。 |

- 高度统一为 **1px**，容器用 `min-height: 1px` 或 `height: 1px` 保证像素对齐。

### 3.3 小竖线（垂直）尺寸与颜色

| 属性 | 取值 | 说明 |
|------|------|------|
| **宽度** | `1px` | 与线宽一致。 |
| **高度** | `16px` | 设计稿固定高度，用于列表项、标签组等。 |
| **线色** | `#C4C7CA` | Figma 小竖线专用色（Line 338），**与水平分割线色 #f2f3f4 不同**，实现时须单独设置。 |
| **实现** | 水平 1px 线旋转 90°，或直接绘制 1×16px 竖线 | 颜色必须使用 #C4C7CA，不可与分割线共用同一 token。 |

## 4. 使用规范（场景与布局）

### 4.1 默认用途说明

- **对不同章节的文本段落进行分割，默认为水平分割线。**
- 即：分割线的主要用途是**段落与段落、章节与章节**之间的视觉分隔，默认形态为水平线。

### 4.2 正确用法（推荐）

| 要点 | 说明 |
|------|------|
| **放置位置** | 放在**两个完整段落/章节块之间**，而不是标题与正文之间。 |
| **上下间距** | 分割线上下应有足够留白；设计稿示例中段落块与分割线之间为 **24px**（`gap: 24px`），保证可读性。 |
| **语义** | 仅对「需要区隔」的内容使用，且分类逻辑正确。 |

**正确示例结构：**

```
[段落块 1：标题 + 正文]
    ↓ 24px
[水平分割线 1px]
    ↓ 24px
[段落块 2：标题 + 正文]
```

- 设计稿中「不同触发方式」正确示例：两个「Default」段落块之间为一条水平分割线，分割线两侧为段落内容，而非紧贴标题下方。

### 4.3 错误用法（避免）

| 要点 | 说明 |
|------|------|
| **错误放置** | 将分割线紧贴在小标题下方、正文上方，用于「标题—分割线—正文」的布局，会混淆层级与阅读顺序。 |
| **错误分类** | 分隔不需要区隔的信息，或分类不当，导致分割线失去语义。 |
| **规范提示** | 「分隔需要进行区隔的信息，分类要正确。」 |

**错误示例结构（避免）：**

```
[小标题 Default]
    ↓ 分割线紧贴标题
[水平分割线 1px]
    ↓
[正文段落]
```

- 设计稿中「不同触发方式」错误示例：分割线出现在标题（Default）与正文之间，且上下间距过紧（如 46px 内包含标题、分割线、正文），不符合「段落间分割」的用法。

### 4.4 极端原则（带文字分割线）

若分割线**带文字**（如「或」「更多」等）：

| 规则 | 说明 |
|------|------|
| **长度** | 文本**不可超出**分割线可视长度。 |
| **字数** | **最好限制在 3 个单词内**（中文建议对应为少量短词），保持简洁。 |
| **中间文案样式** | **无背景色**。中间文字直接叠在容器背景上，不设 `background`，以 1px 线为主视觉。 |

- 带文字分割线为特殊形态，仍以 1px 线 + 两侧短线 + 中间文案呈现；文案过长会破坏视觉平衡，需严格限制。

## 5. 与文档/标题的搭配

### 5.1 文档内标题分割线（Level 1/Light）

- 设计稿中「设计规范」「使用规范」「交互热区」等**文档标题**下方使用 **Level 1/Light** 分割线。
- **线宽**：1px。**线色**：与 `--color-line-border-subtle` 或 Level 1/Light 令牌一致。
- **用途**：区分标题与正文区块，不替代内容区「段落与段落」之间的分割线。

### 5.2 默认样式展示

- 「默认样式」区块中展示的为**水平分割线**：宽度 335px / 343px，高度 1px，浅灰色，无文字，作为组件默认形态的参考。

## 6. 实现要点汇总

### 6.1 必须遵守

| 项目 | 规范 |
|------|------|
| 线宽 | 全部 1px，无例外。 |
| 方向 | 默认水平；垂直仅用「小竖线」且高度 16px。 |
| 位置 | 用于段落/章节之间，且上下留白（如 24px）。 |
| 带文字 | 文本不超出线长，建议 ≤3 个单词；中间文案无背景色。 |

### 6.2 可选扩展

| 项目 | 说明 |
|------|------|
| 类型 | 若设计系统有 Primary / Secondary / Faded / Highlight / Critical，线色可绑对应 token，线宽仍 1px。 |
| 宽度 | 水平分割线宽度随父容器或内容区，不强制 335/343px。 |

### 6.3 设计令牌参考

| 用途 | Token / 值 |
|------|------------|
| 分割线（水平）、边框 | `--color-line-border-subtle`（#f2f3f4） |
| **小竖线（垂直）** | **#C4C7CA**（Figma Line 338，须单独使用，不可与水平分割线共用） |
| 标题/正文主色 | `--color-text-text-primary`（#070808） |
| 正文次级色 | `--text/t-2`（#434446） |
