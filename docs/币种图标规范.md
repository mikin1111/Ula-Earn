## 1. 概述

币种图标用于金融场景下的币种识别与展示，如交易对、资产详情、币种列表等。本规范定义币种图标及链图标的尺寸、间距、占位规则与描边样式。

**核心原则**：
- 统一 Web 与 H5 视觉表现
- 保证币种识别一致性
- 支持单币种与带链币种两种展示形态

## 2. 图标尺寸

### 2.1 主币种图标

| 尺寸 | 像素 | 说明 |
|------|------|------|
| 40 | 40px | 最大，用于重点展示 |
| 36 | 36px | |
| 32 | 32px | 常用 |
| 28 | 28px | |
| 24 | 24px | |
| 20 | 20px | |
| 16 | 16px | 最小，紧凑场景 |

### 2.2 链图标

- **多行**：链图标与主币种同尺寸
- **单行多链**：链图标叠加于主币种右下角，尺寸见 [3.3.1](CoinTitle.md#331-单行多链-web-样式与尺寸)

---

## 3. 带链币种图标布局与间距

**单行多链**：链图标叠加于主币种右下角，尺寸见 [Coin Title 规范 3.3.1](CoinTitle.md#331-单行多链-web-样式与尺寸)。

**多行**：链图标与主币种并排，间距如下：

| 布局类型 | 主币种与链图标间距 |
|----------|-------------------|
| multiple-row | 6px |
| coin-chain-multiple-row | 4px |

---

## 4. 占位图标规则

| 原则 | 说明 |
|------|------|
| 第一原则 | 币种名称首字母占位（如 BTC → B、ETH → E） |
| 第二原则 | 获取不到币种名称时展示通用占位图 |
| 带链占位 | 币种带链时，主图标与链图标均可使用占位 |

占位图标使用与主图标相同的尺寸与圆角，背景色可选用品牌色或中性色（如 BTC 橙 `#F7931A` 作为占位示例）。

---

## 5. 描边样式

| 类型 | 说明 | 适用场景 |
|------|------|----------|
| line-border-subtle | 1px 浅色描边 | Web 端常用 |
| none | 无描边 | H5 或简洁场景 |

**实现**：通过 `border="subtle" | "none"` 控制。使用伪元素绘制描边，`img` 填满整个图标区域，保证币种图标在描边内部完整填充。主币种图标与链图标均应用描边。

---

## 6. 圆角与形状

- 币种图标与链图标均为**圆形**（`border-radius: 50%`）
- 占位图标同样为圆形

---

## 7. 颜色与主题

| 模式 | 说明 |
|------|------|
| Light | 白底，图标与文字使用品牌色/中性色 |
| Dark | 黑底，自动反色适配，保持对比度 |

通过 `data-theme="light" | "dark"` 或父级 `colorMode` 控制。

---

## 8. 与 Coin Title 组件的关系

币种图标规范是 **Coin Title** 组件的子规范。Coin Title 组件负责：
- 币种图标 + 币种名称的排版
- 图标尺寸、间距、占位、描边的实现

详见 `docs/CoinTitle.md`。
