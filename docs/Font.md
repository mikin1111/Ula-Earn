# 字阶组件规范（Figma 设计稿标准）

## 文档说明

本规范根据 Figma 设计稿 **Text Guidelines - Web**（测试V5.1 Web-H5 组件库，节点 134777-4867）**逐项提取**，精准描述不同语言使用的字体、不同功能使用的字号与字重、单行/多行及段落行高、字间距等，供开发与设计对齐使用。

**设计稿来源：**  
[Figma - Text Guidelines Web（字阶）](https://www.figma.com/design/yyvB1Mer7aG8cJx9JVTVZL/%E6%B5%8B%E8%AF%95V5.1-Web-H5%E7%BB%84%E4%BB%B6%E5%BA%93?node-id=134777-4867)

**预览页：** [font.html](../html/font.html)

**最后更新：** 2025-03-15（字体使用方式、H1～H5、字重 400/500/600/700 已同步）

## 1. 字体家族 (Font Family) — 按语言区分

### 字体使用方式

| 语言/场景 | 字体来源 | 字重 (Weights) | 说明 |
|-----------|----------|----------------|------|
| **英文 (EN)** | **Font/Switzer** | 400、500、600、700 | 引用项目内 `Font/Switzer` 中的 Gate-Switzer-*.ttf。 |
| **中文 (CN)** | **Font/苹方** | 400、500、600、700 | 引用项目内 `Font/苹方` 中的 PingFang *.ttf。 |
| **其他语言** | **系统字体** | 400、500、600、700 | 使用系统默认字体，不引用 Font 文件夹。 |

- **字重仅包含：** 400、500、600、700（对应 Regular、Medium、Semibold、Bold）。  
- 设计稿中所有字阶的 **字间距 (Letter Spacing)** 均为 **0**（无额外 letter-spacing）。

---

## 2. 字号功能区分 (Type Scale Roles)

为更好创建信息层次结构并以清晰有效的方式组织内容，字号按功能分为 **4 个层级**：

| 层级 | 用途 | 典型场景 |
|------|------|----------|
| **Display** | 首焦大标题，吸引用户注意力 | 引导页、落地页主标题、标志性标题 |
| **Heading** | 分割内容、引导视觉层次 | 页面标题、模块标题（多级） |
| **Subtitle** | 位于 Heading 之下 | 卡片标题、子模块标题、列表标题、重要信息提示 |
| **Body** | 正文内容 | 段落、列表、说明文字、描述性文本、小字提示等 |

- **Display**：2 档（D1、D2）。  
- **Heading**：5 档（H1～H5）。  
- **Subtitle**：多档（S1～S7），展示不同级别。  
- **Body**：单行正文（B1～B14）与多行正文（BodyMulti / BM1～BM4）。

## 3. Web 端字阶规范 (Text Guidelines - Web)

以下为 **Web** 端全部字阶的精确参数。行高为相对字号的比例，实现时建议使用无单位倍数（如 `line-height: 1.3`）或百分比（如 130%）。Figma 中行高约为 `1.2999999523162842`，统一按 **130%** 表述；Display 的 D1/D2 单独标注。

### 3.1 Display（展示型标题）

| Token 名称 | 字重 | 字号 | 行高 | 字体 | 说明 |
|------------|------|------|------|------|------|
| **D1** | 600 (Semibold) | 80px | **110%** (1.1) | Switzer | 首焦大标题，最大字号 |
| **D2** | 600 (Semibold) | 56px | **120%** (1.2) | Switzer | 次级展示标题 |

- **字间距：** 0。

### 3.2 Heading（标题）— H1～H5

设计稿中展示 1～5 级标题，字重与行高一致，仅字号递减以区分层级。

| Token 名称 | 字重 | 字号 | 行高 | 字体 | 说明 |
|------------|------|------|------|------|------|
| **H1** | 600 (Semibold) | 48px | 130% | Switzer | 一级标题 |
| **H2** | 600 (Semibold) | 40px | 130% | Switzer | 二级标题 |
| **H3** | 600 (Semibold) | 32px | 130% | Switzer | 三级标题 |
| **H4** | 600 (Semibold) | 28px | 130% | Switzer | 四级标题 |
| **H5** | 600 (Semibold) | 24px | 130% | Switzer | 五级标题 |

- 行高统一 **130%**（实现可用 `line-height: 1.3`），字重均为 **600 (Semibold)**，字间距 **0**。

### 3.3 Subtitle（副标题）

设计稿展示 S1、S3～S7；S2 与 S1 同字号不同语义时可按 500 24px 130% 使用。

| Token 名称 | 字重 | 字号 | 行高 | 字体 | 说明 |
|------------|------|------|------|------|------|
| **S1** | 500 (Medium) | 24px | 130% | Switzer | 副标题一 |
| **S2** | 500 (Medium) | 24px | 130% | Switzer | 副标题二（与 S1 同规格，语义区分） |
| **S3** | 600 (Semibold) | 20px | 130% | Switzer | 副标题三 |
| **S4** | 500 (Medium) | 20px | 130% | Switzer | 副标题四 |
| **S5** | 600 (Semibold) | 18px | 130% | Switzer | 副标题五 |
| **S6** | 500 (Medium) | 18px | 130% | Switzer | 副标题六 |
| **S7** | 600 (Semibold) | 16px | 130% | Switzer | 副标题七 |

- 行高统一 **130%**，字间距 **0**。

### 3.4 Body（单行正文）

共 14 档（B1～B14），用于不同级别的正文与说明；同一字号下通过字重区分语义。

| Token 名称 | 字重 | 字号 | 行高 | 字体 | 说明 |
|------------|------|------|------|------|------|
| **B1** | 400 (Regular) | 20px | 130% | Switzer | 正文最大 |
| **B2** | 400 (Regular) | 18px | 130% | Switzer | |
| **B3** | 500 (Medium) | 16px | 130% | Switzer | |
| **B4** | 500 (Medium) | 16px | 130% | Switzer | |
| **B5** | 400 (Regular) | 16px | 130% | Switzer | |
| **B6** | 600 (Semibold) | 14px | 130% | Switzer | |
| **B7** | 500 (Medium) | 14px | 130% | Switzer | |
| **B8** | 400 (Regular) | 14px | 130% | Switzer | |
| **B9** | 600 (Semibold) | 12px | 130% | Switzer | |
| **B10** | 600 (Semibold) | 12px | 130% | Switzer | |
| **B11** | 500 (Medium) | 12px | 130% | Switzer | |
| **B12** | 500 (Medium) | 12px | 130% | Switzer | |
| **B13** | 400 (Regular) | 12px | 130% | Switzer | |
| **B14** | 400 (Regular) | 12px | 130% | Switzer | |

- 行高统一 **130%**，字间距 **0**。B3/B4、B9/B10、B11/B12、B13/B14 为同字号同行高、不同语义或字重区分，按设计 token 选用。

### 3.5 BodyMulti（多行正文）

适用于**长文本**（文章、公告、博客、新手教程等）的**固定行高规范**，与单行 Body 区分。Figma 中组件名为 BodyMulti（拼写为 BodyMutil 的以 BodyMulti 为准）。

| Token 名称 | 字重 | 字号 | 行高 | 字体 | 说明 |
|------------|------|------|------|------|------|
| **BM1** | 400 (Regular) | 16px | **150%** (1.5) | Switzer | 多行正文 16px |
| **BM2** | 400 (Regular) | 14px | **150%** (1.5) | Switzer | 多行正文 14px |
| **BM3** | 400 (Regular) | 12px | **150%** (1.5) | Switzer | 多行正文 12px |
| **BM4** | 500 (Medium) | 14px | 130% | Switzer | 多行正文 14px 中等强调 |

- BM1～BM3 行高为 **150%**（`line-height: 1.5`），便于长文阅读。  
- BM4 行高为 130%，与单行 Body 一致。

### 3.6 多行、段落字体行高（按字号）

设计稿中「多行、段落字体行高」一节明确：**固定行高规范仅面向长文本（文章/公告/博客/新手教程）**，不用于单行 UI 文案。

| 字号 | 行高 | 行高比 | 说明 |
|------|------|--------|------|
| **16px** | **150%** | 1.5 | 16px 段落标准行高 |
| **14px** | **170%** | 约 1.7 | 14px 段落标准行高 |
| **12px** | **180%** | 1.8 | 12px 段落标准行高 |

- 实现时可用 `line-height: 1.5`、`1.7`、`1.8`。  
- 与 BodyMulti（BM1～BM3）配合使用：BM1 对应 16px/150%，BM2 对应 14px/150%（若需更松可选用 170%），BM3 对应 12px/150%（若需更松可选用 180%），以产品与设计约定为准。

## 4. 字重 (Font Weight) — 按语言与功能区分

设计稿分别给出 **英文 EN** 与 **中文 CN** 的字重说明，**不同语言可用字重略有差异**。

### 4.1 英文 EN：Switzer

| 字重数值 | 样式名 | 说明 |
|----------|--------|------|
| 400 | Regular | 常规正文 |
| 500 | Medium | 中等强调 |
| 600 | Semibold | 半粗，标题/副标题常用 |
| 700 | Bold | 粗体强调 |

- 设计稿字体家族写 **400、500、600、700**；字重示例区仅展示 400、500、600 三种示例，实际规范以 400/500/600/700 为准。

### 4.2 中文 CN：苹方 (PingFang SC)

| 字重数值 | 样式名 | 说明 |
|----------|--------|------|
| 400 | Regular | 常规正文 |
| 500 | Medium | 中等强调 |
| 600 | Semibold | 半粗（若字体无 600，可用 700 替代） |
| 700 | Bold | 粗体强调 |

- 字重与英文一致，**仅使用 400、500、600、700**，字体文件来自 Font 文件夹。

### 4.3 小结

| 语言 | 可用字重 | 对应样式名 |
|------|----------|------------|
| **英文 (Switzer)** | 400、500、600、700 | Regular、Medium、Semibold、Bold |
| **中文 (苹方)** | 400、500、600、700 | Regular、Medium、Semibold、Bold |
| **其他语言 (系统字体)** | 400、500、600、700 | Regular、Medium、Semibold、Bold |

- 全语言字重**仅包含** 400、500、600、700，不使用其他字重。

## 7. 实现参考 (CSS / 开发)

- **行高：** 建议使用无单位倍数（如 `line-height: 1.3`），便于随字号缩放。  
- **字号：** 若需 rem 适配，可按 1rem = 16px 换算（如 20px = 1.25rem）。  
- **字重：** 仅使用 `font-weight: 400 | 500 | 600 | 700`。  
- **字间距：** `letter-spacing: 0` 或 normal。  
- **字体使用方式：**  
  - **英文：** 引用 **Font/Switzer** 中的 Switzer，`font-family: 'Switzer', sans-serif`（通过 @font-face 加载 `Font/Switzer/Gate-Switzer-*.ttf`）；  
  - **中文：** 引用 **Font/苹方** 中的苹方 (PingFang SC)，`font-family: 'PingFang SC', sans-serif`（通过 @font-face 加载 `Font/苹方/PingFang *.ttf`）；  
  - **其他语言：** 使用系统字体，`font-family: system-ui, sans-serif`，不引用 Font 文件夹。
