Header / Footer 组件使用指南

本文档介绍 Gate 设计规范下的 Header 与 Footer 组件用法，适用于 Web 与 H5 双端。

1. 快速开始

1.1 引用资源

组件依赖以下资源，请确保路径正确：

类型 路径 说明
切图 Slicing/Header/Logo.svg 品牌 Logo
切图 Slicing/Header/QR Code.svg Footer 下载区二维码
图标 icons/Basic%20Product%20Icons/*.svg 功能图标
图标 icons/Social%20Media%20Icons/*.svg 社媒图标

1.2 任务清单

实现前请确认：

  • 已引入 color-tokens.css 或等效颜色变量
  • 已引入 Switzer 字体（见 Font.md）
  • 已配置 Select 下拉组件（Footer 语言/货币）
  • 已实现登录态切换逻辑

2. 布局规格

2.1 Header 容器

属性 Web H5
高度 48px 48px
背景 --bg-primary 同左
水平内边距 24px 16px

2.2 Footer 容器

属性 取值
背景 --bg-secondary
上内边距 80px
内容区最大宽度 1200px
列间距 24px

3. 代码示例

3.1 基础 HTML 结构

<header class="earn-header">
  <img class="earn-header__logo" src="../Slicing/Header/Logo.svg" alt="Logo" />
  <nav class="earn-header__nav">
    <a href="#">Earn</a>
  </nav>
  <div class="earn-header__right">
    <a href="#">Log In</a>
    <button class="btn btn--primary">Sign Up</button>
  </div>
</header>

3.2 CSS 变量引用

/* 背景与文字色 */
.earn-header {
  background: var(--bg-primary);
  color: var(--text-primary);
}

.earn-footer {
  background: var(--bg-secondary);
  padding-top: 80px;
}
